<p><h1 id="_just%20Docs">_just Docs</h1>
Welcome to Just an Ultimate Site Tool documentation!
<span class="OGCa03R">"<strong><code>&&#35;95;just</code></strong>" is an abbreviation of <strong>Just an Ultimate Site Tool</strong>.</span><div class="sBeP7"></div><br><h2 id="What%20is%20_just%3F">What is _just?</h2>
Just an Ultimate Site Tool is a GitHub Action for building static websites. 
Currently it have 4 modes: 
<ul><li><code>Postprocessor</code>: Modify and add your own files to generated Next.js website.</li><li><code>Redirector</code>: Client-side redirect using JavaScript.</li><li><code>Compressor</code>: Compresses your website.</li><li><code>Generator</code>: Generates documentation website using Markdown.</li></ul>

<blockquote class="sBePB">Just an Ultimate Site Tool is still in development at the <strong>alpha</strong> stage. Expect regular updates, possible bugs, and changes. If you have found a bug, please <a href="https://github.com/js-just/_just/issues/new?labels=bug&template=bug.md" target="_blank" id="kjwDV">report it here</a>.</blockquote>
<blockquote class="sBePL">The docs are under construction!</blockquote>
<blockquote class="sBePH">Do not use <code>Redirector</code> if you can make server-side <code>HTTP 3XX</code> redirects.</blockquote>
<h2 id="Why%20_just%3F">Why _just?</h2>
<ol><li><u>No packages.</u></li><li><u>Fast build.</u></li><li><strong class="sBeP">No watermarks.</strong></li></ol>
<span class="OGCa03R">_just uses Node.js, but _just does not require you to use Node.js/npm/pnpm/Yarn/related packages stuff.</span>
