This mode requires the <code>just&#46;config&#46;js</code> file and the workflow file.

After generating the documentation, this mode uses the <a href="/docs/modes/compressor" target="_self">Compressor mode</a> to compress the generated website.
Use Markdown (`.md`) files for documentation. You can also use HTML/CSS/JavaScript for custom pages, but remember that they will be compressed using the <a href="/docs/modes/compressor" target="_self">Compressor mode</a>!

<h2 id="How%20it%20works%3F2">How it works?</h2>
It processes every Markdown file and generates HTML page for each of them.
