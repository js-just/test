<h2>What is _just?</h2>
Just an Ultimate Site Tool is a GitHub Action for building static websites. 
Currently it have 4 modes: 
- `Postprocessor`: Modify and add your own files to generated Next.js website.
- `Redirector`: Client-side redirect using JavaScript.
- `Compressor`: Compresses your website.
- `Generator`: Generates documentation website using Markdown.

<blockquote class="sTWUB">Just an Ultimate Site Tool is still in development at the <strong>alpha</strong> stage. Expect regular updates, possible bugs, and changes. If you have found a bug, please <a href="https://github.com/js-just/_just/issues/new?labels=bug&template=bug.md" target="_blank" id="rGbS9">report it here</a>.</blockquote>
<blockquote class="sTWUE">The docs are under construction!</blockquote>
<blockquote class="sTWU0">Do not use <code>Redirector</code> if you can make server-side <code>HTTP 3XX</code> redirects.</blockquote>
<h2>Why _just?</h2>
1. __No packages.__
2. __Fast build.__
3. __**No watermarks.**__
<span class="loueCmc">_just uses Node.js, but _just does not require you to use Node.js/npm/pnpm/Yarn/related packages stuff.</span>
